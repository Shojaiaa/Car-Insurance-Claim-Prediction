# Car Insurance Claim Prediction

## Project Overview

This machine learning project aims to predict whether a car insurance customer will make a claim using demographic, vehicle, and driving-related information.

The dataset contains 10,000 customer records. The target is `OUTCOME`:

- `0`: The customer did not make an insurance claim.
- `1`: The customer made an insurance claim.

## Project Status

This project is currently in progress. Part 1 includes:

- Data exploration and cleaning.
- Exploratory visualizations.
- Data preprocessing.
- Training and evaluating a default classification model.
- Identifying important features using permutation importance.
- Creating explanatory visualizations for two important features.

Additional modeling and project updates will be added in the upcoming parts.
